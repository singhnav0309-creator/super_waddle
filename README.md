# Flask-REGISTRATION Website

 A simple flask web application using a Bootstrap and Navbar.

 ## CHARACTERISTICS
 - Flask routing
 - Home Page
 - HTML templates
 - static file
 - Bootstrap Navbar
 - Python source code

## SOFTWARES USED
- Python
- Flask
- HTML
- Bootstrap

## Installation
* Clone respository
  '''
  git clone https://github.com/singhnav0309-creator/super_waddle.git
  '''

* Open Project folder
  '''
  cd super_waddle
  '''

* Install Flask
  '''
  pip install flask
  '''
  
* Run the Project
  '''
  python new.py
  '''

* Open Browser
  '''
  http://127.0.0.1:5000
  '''
